Builders Night Berlin #2 - 26th of February

Early-20s hackers sat next to founders of Berlin’s top tech startups at our first OpenClaw Builders Night, all tinkering away together.

Ever wished you could see why the Gemini model chose one word over another? Now you can. The model's decision-making process is no longer a black box. The #Logprobs is now officially re-introduced in the Gemini API on Vertex AI, unlocking the model's reasoning.

This feature transforms the model's output from a simple answer into a transparent, quantifiable decision, letting you see the probability scores for the chosen token and its top alternatives.

This isn't just about debugging; it’s about building smarter, more reliable applications. Here are a few powerful example use cases you can now do:

✅ Smarter Classification
↳ For any classification, you get a quantifiable score. Now you can programmatically decide whether to trust an output, or flag it for human review if the model is uncertain.

✅ Better Autocomplete
↳ You can build dynamic autocomplete that suggests multiple, ranked next-word options, adapting as the user types.

✅ RAG Evaluation
↳ You can calculate an average logprobs score for a generated answer to measure how "grounded" it is in the context you provided. It’s a great way to quantitatively assess the quality of your RAG pipeline.

These are just a few of the example applications, and with this feature, there are many more use cases for developers to explore.

If you want to see the code for these examples, check out the introductory notebook: https://lnkd.in/gDTPKk27
